# B-MEDICS Website

This site is a responsive one-page website for B-MEDICS.

## Files
- B-Medics.html
- style.css
- js/script.js
- thank-you.html

## Deployment options

### GitHub Pages
1. Create a GitHub repository.
2. Upload these files.
3. In Settings > Pages, select the main branch and root folder.
4. Your site will be published at your GitHub Pages URL.

### Netlify
1. Drag and drop the folder into Netlify.
2. The site will be published automatically.

### Vercel
1. Import the folder into Vercel.
2. Deploy it.

### Custom domain
Use the domain name: bmedics.co.za
